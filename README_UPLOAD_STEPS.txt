SUDAQ Verification Website Files

Upload these files to your GitHub repo root:
- index.html
- events.json
- logo.jpg
- all certificates-*.json files

Steps:
1. Open your GitHub repo: sudaq-verify
2. Delete the old index.html if needed.
3. Upload all files from this ZIP to the repo root.
4. Commit changes.
5. Open: https://sudaq-verification.github.io/sudaq-verify/
6. Test using SUDV-001.

Important:
- The website reads events.json.
- Every JSON file listed in events.json must be uploaded to the repo root.
- To add a future event, upload its certificates-*.json file and add its filename to events.json.
